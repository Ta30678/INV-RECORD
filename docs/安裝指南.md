# 安裝指南

## 需求

- [Obsidian](https://obsidian.md/) 1.5 以上（桌面版；行動版理論上可用但未驗證）

## 方式一：直接使用 vault 模板（推薦）

repo 內的 `vault-template/` 是一個完整可用的 vault，外掛已安裝好：

1. 把 `vault-template/` 複製到你想放筆記的位置，並改成喜歡的名字（例如 `我的投資筆記`）
2. 打開 Obsidian → **開啟資料夾作為儲存庫**（Open folder as vault）→ 選剛複製的資料夾
3. 第一次開啟會詢問是否信任作者並啟用社群外掛 → 選擇**信任並啟用**（外掛需要啟用才能顯示 K 線圖與儀表板）
   - 若錯過提示：設定 → 第三方外掛 → 關閉「限制模式」→ 啟用「INV Record 投資紀錄」
4. 打開 `30-個股/台積電 2330`，應該會看到 K 線圖載入
5. `Ctrl/Cmd + P` → 「INV Record: 開啟績效儀表板」，右側欄會出現持倉與損益

## 方式二：安裝到現有 vault

1. 在 repo 的 `plugin/` 目錄執行：

   ```bash
   npm install
   npm run build
   npm run install:vault /path/to/你的vault
   ```

   或手動把 `plugin/` 下的 `main.js`、`manifest.json`、`styles.css` 三個檔案
   複製到 `你的vault/.obsidian/plugins/inv-record/`

2. Obsidian 設定 → 第三方外掛 → 重新載入後啟用「INV Record 投資紀錄」
3. 到外掛設定頁調整「交易紀錄資料夾」等路徑，以符合你 vault 的結構

## 開發

```bash
cd plugin
npm install
npm test          # 單元測試（FIFO、Yahoo 解析、費用試算…）
npm run build     # 型別檢查 + 打包 main.js
npm run dev       # watch 模式
npm run check:yahoo   # 選跑：實測 Yahoo endpoint
```
