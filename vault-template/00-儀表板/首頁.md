# 投資紀錄 首頁

歡迎使用 INV RECORD。這個 vault 以三層架構組織投資思考：

> **總經 → 題材 → 個股／交易**

## 入口

- [[總經 MOC]] — 國際局勢、央行利率決策
- [[題材 MOC]] — 產業題材與領域
- 個股筆記在 `30-個股/`，例如 [[台積電 2330]]
- 交易紀錄在 `40-交易紀錄/`

## 常用操作

- `Ctrl/Cmd + P` → 「INV Record: 新增交易紀錄」— 記一筆買賣（自動試算手續費/證交稅）
- `Ctrl/Cmd + P` → 「INV Record: 開啟績效儀表板」— 看持倉與已實現損益
- 左側緞帶的 📈 圖示也能開儀表板
- 個股筆記裡的 ` ```kline ` 區塊會自動載入 Yahoo 的日/週/月 K 線圖

## 關係圖

打開 Graph View（`Ctrl/Cmd + G`）可以看到三層結構：

- 🔵 總經　🟣 題材　🟠 個股　🔴 交易紀錄
