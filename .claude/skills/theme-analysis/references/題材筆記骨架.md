## B. 四張標準圖表 invchart 範例

只在有查證資料時畫；缺資料就整張略過。放在對應資料表格**前面**（圖前表後）。以下數字為格式示意，務必替換。

### B1. 市場規模逐年（bar）

```invchart
type: bar
title: <題材> 市場規模（美元）
labels: [2024, 2025, 2026(e), 2027(e), 2030(e)]
series:
  - name: 市場規模
    data: [46, 55, 63, 72, 79]
unit: 億美元
source: <來源機構 發布日期>（區間值取中點，精確值見下表）
asOf: <YYYY-MM>
```

### B2. 需求／出貨成長（line；也可用 bar）

```invchart
type: line
title: <題材> 需求量逐年
labels: [2025, 2026(e), 2027(e)]
series:
  - name: 需求量
    data: [68.9, 139.4, 269.4]
unit: 萬片
source: <來源機構 發布日期>
asOf: <YYYY-MM>
```

量＋成長率想同圖時改用 bar-line（每個 series 要標 `type`，成長率放右軸；`data` 不可留空/null，長度須等於 labels，起始年若無 YoY 就從有值的年份起算）：

```invchart
type: bar-line
title: <題材> 出貨量與年增率
labels: [2025, 2026(e), 2027(e)]
series:
  - name: 出貨量
    type: bar
    data: [68.9, 139.4, 269.4]
  - name: YoY
    type: line
    yAxis: right
    data: [24, 102, 93]
unit: 萬片
unitRight: "%"
source: <來源機構 發布日期>
asOf: <YYYY-MM>
```

### B3. 產能擴張逐年（bar；區間取中點）

```invchart
type: bar
title: <題材/廠商> 月產能擴張
labels: [2024年底, 2026年中, 2026年底(目標), 2027年底(目標)]
series:
  - name: 月產能
    data: [3.5, 7.75, 12.5, 20]
unit: 萬片/月
source: <來源機構 發布日期>（區間值取中點，精確區間見下表）
asOf: <YYYY-MM>
```

### B4. 買方或市佔分配（pie；≤7 項，單一時點）

```invchart
type: pie
title: <題材> 產能/需求分配（<年份>）
labels: [Nvidia, Broadcom, AMD, 世芯-KY, 其他]
series:
  - name: 佔比
    data: [60, 15, 11, 6, 8]
unit: "%"
source: <來源機構 發布日期>
asOf: <YYYY 或 YYYY Qn>
```

## C. 圖表選型速查（摘自 docs/圖表語法.md，以原文為準）

| 資料形態 | 圖型 |
|---|---|
| 佔比／組成（單一時點，≤7 項） | `pie` / `donut` |
| 佔比 >7 項或類別名很長 | `bar` + `horizontal: true` |
| 趨勢（時間序列） | `line` |
| 離散期間比較（年度金額/產能） | `bar` |
| 量＋率 雙軸 | `bar-line`（右軸放率，series 標 type） |
| 結構隨時間變化 | `bar` + `stacked: true` |

必填欄位：`type`、`title`、`labels`、`series[].name`、`series[].data`、`source`、`asOf`；`series[].data` 長度須等於 `labels` 長度。
