## 四張標準圖（invchart，有資料才畫，圖前表後）

圖型選擇與完整欄位以 `docs/圖表語法.md` 為準。`labels` 與每個 `series[].data` 長度必須相等；`source`／`asOf` 必填；中性資料用內建色盤，勿套紅漲綠跌。

### 1. 營收結構（pie，單一時點、≤7 項）
````markdown
```invchart
type: pie
title: [公司] 營收結構（[時點]）
labels: [HPC, 智慧型手機, IoT, 車用, 其他, 消費性電子]
series:
  - name: 營收佔比
    data: [61, 26, 6, 4, 2, 1]
unit: "%"
source: [公司 法說會／來源]
asOf: [2026 Q1]
```
````

### 2. 月營收＋YoY（bar-line，量＋率雙軸）
````markdown
```invchart
type: bar-line
title: [公司] 月營收與年增率
labels: [2026-01, 2026-02, 2026-03, 2026-04, 2026-05, 2026-06]
series:
  - name: 月營收
    type: bar
    data: [2932, 2600, 2853, 3495, 3205, 3170]
  - name: YoY
    type: line
    yAxis: right
    data: [35.9, 43.1, 46.5, 48.1, 39.6, 26.9]
unit: 億元
unitRight: "%"
source: [公開資訊觀測站自結月營收／財報狗]
asOf: [2026-06]
```
````

### 3. 毛利率逐季（line，趨勢）
````markdown
```invchart
type: line
title: [公司] 毛利率逐季
labels: [2025 Q2, 2025 Q3, 2025 Q4, 2026 Q1]
series:
  - name: 毛利率
    data: [56.0, 59.6, 59.0, 58.8]
unit: "%"
source: [公司財報／財報狗]
asOf: [2026 Q1]
```
````

### 4. 法人目標價（bar + horizontal，券商多／名稱長）
````markdown
```invchart
type: bar
title: 法人目標價（[發布區間]）
labels: [里昂, 花旗, 匯豐, 野村, 永豐金, 元大/凱基, Aletheia, 高盛]
series:
  - name: 目標價
    data: [3030, 2875, 2820, 2800, 2615, 2600, 2400, 2330]
horizontal: true
unit: 元
source: [TechNews 2026-04／豐雲學堂 2026-06]
asOf: [2026-06]
```
````

### 區間值取中點示範
若來源給「12~13 萬片」：`data` 填 `12.5`，並在 `source` 註明「區間值取中點，精確區間見下表」，圖後表格保留 `12~13 萬片`。

### 其他可用圖型（依 docs/圖表語法.md）
- 結構隨時間變化（各平台營收佔比逐季）→ `bar` + `stacked: true`。
- 離散期間比較（年度市場規模、產能擴張）→ `bar`。
- 少於 3 個資料點、或表格一眼可讀 → 不畫圖。
